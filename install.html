Install / FAQ

How install works

Usually plug-and-play behind the dash with a harness. We provide a guide and help confirm screen type.

Will it look OEM?

Yes — factory screen stays. The module adds CarPlay/AA as an additional input/launcher.

Wireless vs wired

Many setups support wireless CarPlay. Some vehicles may prefer wired for stability — we’ll advise per model.

Fitment checklist (send this):
• Year / Make / Model
• Factory navigation screen? (Yes/No)
• Photo of the screen + center console
• Any aftermarket audio installed? (Yes/No)

Common Questions

Do I lose factory functions?

Typically no. OEM audio/features remain. CarPlay runs as an added interface.

Sound quality?

Depends on model integration. Most customers report clean audio when installed correctly.
